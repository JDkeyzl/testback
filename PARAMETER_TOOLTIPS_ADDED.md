# 参数提示说明功能添加总结

## 🎯 功能概述

为右侧参数输入区的每个参数添加了详细的提示说明，使用 Tooltip 和 HelpIcon 来提供参数解释，帮助用户更好地理解每个参数的作用和用法。

## ✅ 实现内容

### 1. **导入必要组件**
```javascript
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from './ui/tooltip'
import { HelpCircle } from 'lucide-react'
```

### 2. **创建参数帮助组件**
```javascript
// 参数说明组件
const ParameterHelp = ({ description }) => (
  <TooltipProvider>
    <Tooltip>
      <TooltipTrigger asChild>
        <HelpCircle className="h-4 w-4 text-muted-foreground hover:text-foreground cursor-help" />
      </TooltipTrigger>
      <TooltipContent>
        <p className="max-w-xs text-sm">{description}</p>
      </TooltipContent>
    </Tooltip>
  </TooltipProvider>
)
```

### 3. **为所有参数类型添加帮助说明**

#### **移动均线 (MA) 参数**
- **周期**: "用于计算移动平均线的时间窗口长度，单位：天。数值越大，均线越平滑，反应越慢；数值越小，均线越敏感，反应越快。"
- **阈值**: "用于触发买入/卖出信号的数值条件。当移动均线价格与阈值满足比较条件时，将触发相应的交易信号。"
- **操作符**: "决定比较逻辑的运算符。例如：大于(>)表示当移动均线价格大于阈值时触发信号，小于(<)表示当价格小于阈值时触发信号。"

#### **价格区间参数**
- **最低价**: "价格区间的下限值。当股票价格低于此值时，将触发相应的交易信号。通常用于支撑位或买入信号。"
- **最高价**: "价格区间的上限值。当股票价格高于此值时，将触发相应的交易信号。通常用于阻力位或卖出信号。"

#### **RSI指标参数**
- **周期**: "用于计算RSI指标的时间窗口长度，单位：天。RSI是相对强弱指标，用于判断股票的超买超卖状态。常用周期为14天。"
- **RSI阈值**: "RSI指标的触发阈值。RSI值范围0-100，通常30以下为超卖区域（买入信号），70以上为超买区域（卖出信号）。"
- **操作符**: "决定RSI比较逻辑的运算符。例如：小于(<)表示当RSI小于阈值时触发信号（超卖买入），大于(>)表示当RSI大于阈值时触发信号（超买卖出）。"

#### **布林带参数**
- **周期**: "用于计算布林带中轨（移动平均线）的时间窗口长度，单位：天。布林带由中轨、上轨和下轨组成，用于判断价格波动范围。"
- **标准差**: "布林带的标准差倍数，用于计算上轨和下轨。标准差越大，布林带越宽，价格波动范围越大。常用值为2倍标准差。"

#### **MACD指标参数**
- **快线周期**: "MACD指标中快线（EMA）的计算周期，单位：天。MACD由快线、慢线和信号线组成，用于判断趋势变化和买卖信号。"
- **慢线周期**: "MACD指标中慢线（EMA）的计算周期，单位：天。慢线周期通常大于快线周期，用于平滑价格波动，常用值为26天。"
- **信号线周期**: "MACD指标中信号线（EMA）的计算周期，单位：天。信号线是MACD线的移动平均线，用于生成买卖信号，常用值为9天。"
- **阈值**: "MACD指标的触发阈值。当MACD线与信号线的差值超过此阈值时，将触发相应的交易信号。正值表示买入信号，负值表示卖出信号。"
- **操作符**: "决定MACD比较逻辑的运算符。例如：大于(>)表示当MACD差值大于阈值时触发信号，小于(<)表示当MACD差值小于阈值时触发信号。"

#### **成交量和价格参数**
- **阈值**: 
  - 成交量: "成交量的触发阈值，单位：股。当成交量超过此值时，将触发相应的交易信号。通常用于判断市场活跃度。"
  - 价格: "价格的触发阈值，单位：元。当股票价格与阈值满足比较条件时，将触发相应的交易信号。"
- **操作符**:
  - 成交量: "决定成交量比较逻辑的运算符。例如：大于(>)表示当成交量大于阈值时触发信号，小于(<)表示当成交量小于阈值时触发信号。"
  - 价格: "决定价格比较逻辑的运算符。例如：大于(>)表示当价格大于阈值时触发信号，小于(<)表示当价格小于阈值时触发信号。"

#### **逻辑节点参数**
- **逻辑类型**: "决定多个条件之间的逻辑关系。AND(与)：所有条件都满足时才触发；OR(或)：任一条件满足时就触发；NOT(非)：条件不满足时才触发。"

#### **动作节点参数**
- **动作类型**: "选择交易动作类型。买入：执行买入操作；卖出：执行卖出操作；持有：保持当前仓位不变。"
- **数量**: "交易数量，单位：股。买入时表示买入的股票数量，卖出时表示卖出的股票数量。数量越大，交易金额越大。"
- **价格类型**: "选择交易价格类型。市价：以当前市场价格立即成交；限价：设定特定价格，只有达到该价格时才成交。"
